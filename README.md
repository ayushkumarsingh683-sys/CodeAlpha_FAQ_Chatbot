# Intelligent AI FAQ Chatbot

A simple question-and-answer web app built with Python and Streamlit. 

Instead of searching for exact words, this app uses a smart AI model to understand the actual meaning of your question. It compares your input to saved FAQs and instantly displays the best answer.

## 🚀 Key Features
* **Smart Search:** Understands your question even if you use different words or spellings.
* **Easy Interface:** Clean and simple web layout with a clear text input box.
* **Fast Answers:** Loads the AI model quickly to deliver responses instantly.
* **Friendly Warnings:** Safely displays a clean warning message if you ask an unrelated question.

## 🛠️ Built With
* **Python** - Programming language
* **Streamlit** - Web interface framework
* **Sentence-Transformers** - AI model for understanding text meaning
